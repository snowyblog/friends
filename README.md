# 雪的日记 — 友情链接

在这里提交你的博客，加入我们的友链圈。

## 如何添加

1. Fork 本仓库
2. 在 `links/` 目录下新建一个 `.yaml` 文件，文件名随意（建议用你的 ID）
3. 按下方格式填写，提交 Pull Request
4. CI 自动校验格式 + 链接可达性，通过后等待人工合入

## 文件格式

```yaml
name: "博客名称"       # 必填，你的博客名称
description: "博客简介" # 必填，一句话介绍你的博客
url: "https://..."     # 必填，博客首页地址
icon: "https://..."    # 必填，博客图标/头像 URL
```

## 注意事项

- 博客需为个人原创内容，不接受商业站、采集站、空站
- 图标建议使用正方形图片（会被裁剪为圆形展示）
- 提交前请确认你的博客可正常访问
- 如果你的博客新增了本站友链，请在 PR 中注明，会优先合入
