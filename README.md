# 雪的日记 — 友情链接

在这里提交你的博客，加入我们的友链圈。

## 如何添加

1. 进入 [Issues](https://github.com/snowyblog/friends/issues) 页面
2. 点击 **New Issue** → 选择 **友链申请** 模板
3. 填写你的博客信息：名称、网址、简介、图标
4. 提交后 CI 自动校验，通过后自动创建 Pull Request
5. 等待人工审核合入即可

## 友链格式

```yaml
name: "博客名称"       # 必填
description: "博客简介" # 必填
url: "https://..."     # 必填
icon: "https://..."    # 必填，正方形图片为佳
```

## 注意事项

- 博客需为个人原创内容，不接受商业站、采集站、空站
- 图标建议使用正方形图片（展示时会裁剪为圆形）
- 如果你的博客添加了本站友链，会在审核时优先通过
- 如果 Issue 模板无法使用，也可以直接 Fork → 在 `links/` 下添加 YAML → 提 PR
